# LinkedIn-Promoted-Content-Hide-Tool

LinkedIn feed'indeki "Tanıtılan içerik" (sponsored) gönderilerini ve reklam kutularını otomatik gizler.

## Kurulum

1. Tarayıcına [Tampermonkey](https://www.tampermonkey.net/) eklentisini kur
2. [Buraya tıkla](../../raw/main/linkedin-reklam-gizle.user.js) → "Install" butonuna bas
3. LinkedIn'i aç, sol altta rozet görünüyorsa kurulum tamam

## Özellikler
- Feed'deki tanıtılan içerikleri gizler
- Kenar çubuğu reklamlarını gizler  
- Premium satış kutularını gizler
- Sağ alttaki rozetle açıp kapatılabilir
